How to make an API server in Express.

Currently I have this API: Comments API

Application Programming Interfaces is a way to standardise how applications communicate. The above link to my Comments API follows the conventions for API databases. The noun being "Comments", these being (in order)

  • GET ("GET" being the HTTP method verb) /comments ("/comments" being the API endpoint, ie., the URL), which returns all the items in the database,
  • GET /comments/:id (which returns one specific database item via the ":id" parameter),
  • POST /comments (which inserts a new user-defined item into the database, "POST" being the verb responsible for that),
  • PUT /comments/:id (which replaces a specific item with a new one),
  • PATCH /comments/:id (which replaces part of an item),
  • and DELETE /comments/:id (which deletes a specified item).

For example, if you wished to access the comment at the second position in the database, you'd enter the following URL: http://www.mywebsite.com/api/comments/1. Note that you cannot access anything but the two GET verbs via ordinary browser activity -- the other verbs (POST, PUT, etc) require specialised tools to access, such as Postman, or a specially-programmed website interface.

By following the API conventions, along with using the JSON data format, it becomes significantly easier to have databases exchange information, since the "language" is agreed-upon beforehand.
