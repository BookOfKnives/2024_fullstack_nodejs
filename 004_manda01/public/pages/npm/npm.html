The Node Package Manager (NPM) is a Command-Line Interface (CLI) tool used to quickly disseminate pre-packaged code.

It has a lot of functionality, but for me, I use it with the command npm init when starting a new project. This command asks the user some questions and generates a package.json file from it.


This shows the npm init command running, asking me some questions. At the point of this screenshot, I am halfway through the process.
This shows the npm install nodemon command, which tells NPM to install the package called 'Nodemon'. At this point, the command has already run, so it has returned a status report: NPM tells me the package is update to date (because in this project, I have already installed Nodemon).
