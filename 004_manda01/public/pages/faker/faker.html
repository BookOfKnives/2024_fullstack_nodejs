Faker is a Node package that can generate fake details.

This is used to test computer systems using fitting data. For example, suppose you're making a user registration form -- to test it, you have to enter user data, eg., name, password, email etc. Doing this each time you test is tediuos and resource-consuming. So Faker can be used to generate information for these fields automatically.
