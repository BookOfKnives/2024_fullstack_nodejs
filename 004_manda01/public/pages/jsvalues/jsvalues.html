I learned

in JavaScript objects key-value pairs, assignments overwrite based on name.

For example,

const valueObj = {value: 20, value: 100};
console.log("the value of valueObj:", valueObj.value)

will produce

the value of valueObj: 100

This is because the last value-assignment (the one that says value: 100) overwrites the earlier (the value: 20). This is a feature of JavaScript.
